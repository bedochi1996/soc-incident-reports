# SOC Incident Reports Portfolio

This repository contains simulated SOC investigation reports created for portfolio and training purposes.

The reports demonstrate practical Blue Team workflows including alert triage, IOC extraction, MITRE ATT&CK mapping, timeline reconstruction, root cause analysis, containment actions, and remediation recommendations.

---

## Reports

### 01 — Brute Force Investigation
Focus areas:
- Authentication log review
- Failed login analysis
- Suspicious successful login
- MITRE ATT&CK T1110 mapping
- Account containment recommendations

### 02 — C2 Beaconing Investigation
Focus areas:
- Suspicious outbound traffic
- Periodic beaconing pattern
- Malicious IP review
- Command and Control mapping
- Endpoint isolation and blocking recommendations

### 03 — Phishing Investigation
Focus areas:
- Email header review
- Suspicious URL analysis
- IOC extraction
- User click investigation
- Remediation and awareness actions

---

## Methodology

Each report follows a SOC-style structure:

1. Executive Summary
2. Alert Overview
3. Evidence Reviewed
4. Investigation Timeline
5. Indicators of Compromise
6. MITRE ATT&CK Mapping
7. Root Cause Analysis
8. Impact Assessment
9. Response Actions
10. Recommendations

---

## Disclaimer

All reports are based on simulated lab environments. No real customer data, production credentials, or sensitive logs are included.
